# StudyTime — Flutter Migration Plan

Cross-platform rebuild of the native Android Kotlin app using Flutter + Firebase.
Same Firebase backend, same Firestore schema, zero data migration needed.

---

## Architecture Decisions

| Android (current) | Flutter (new) |
|---|---|
| Kotlin | Dart |
| MVVM + ViewModel + StateFlow | Riverpod (`AsyncNotifier` + `AsyncValue<T>`) |
| `Resource<T>` (Loading/Success/Error) | `AsyncValue<T>` built into Riverpod |
| Hilt dependency injection | Riverpod providers (`keepAlive: true`) |
| Jetpack Navigation + nav_graph.xml | GoRouter |
| Fragment + ViewBinding | `ConsumerWidget` (Riverpod-aware widget) |
| RecyclerView + ListAdapter | `ListView.builder` / `GridView.builder` |
| `activityViewModels()` shared state | Global Riverpod provider |
| `suspend fun` + `await()` | `async` / `await` with `Future` |
| Coroutines `Flow` | Dart `Stream` |
| MaterialDatePicker | `showDatePicker()` (Flutter built-in) |
| Snackbar | `ScaffoldMessenger.showSnackBar()` |
| Hilt `@Singleton` | Riverpod `ref.keepAlive()` |

## Riverpod State Pattern

Every screen follows this pattern — `AsyncValue<T>` replaces `Resource<T>`:

```dart
// provider (login_provider.dart)
@riverpod
class LoginNotifier extends _$LoginNotifier {
  @override
  AsyncValue<void> build() => const AsyncData(null);

  Future<void> login(String email, String password) async {
    state = const AsyncLoading();
    state = await AsyncValue.guard(
      () => ref.read(authRepositoryProvider).login(email, password),
    );
  }
}

// screen (login_screen.dart)
ref.listen(loginNotifierProvider, (_, next) {
  next.whenOrNull(
    error: (e, _) => ScaffoldMessenger.of(context)
        .showSnackBar(SnackBar(content: Text(e.toString()))),
    data: (_) => context.go('/dashboard'),
  );
});
```

## GoRouter Auth Redirect

```dart
redirect: (context, state) {
  final isLoggedIn = FirebaseAuth.instance.currentUser != null;
  final isAuthRoute = state.matchedLocation.startsWith('/login') ||
                      state.matchedLocation.startsWith('/register') ||
                      state.matchedLocation.startsWith('/reset-password');
  if (!isLoggedIn && !isAuthRoute) return '/login';
  if (isLoggedIn && isAuthRoute) return '/dashboard';
  return null;
},
```
